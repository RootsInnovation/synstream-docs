# Sidebar - Git Version Control

The Git Version Control panel provides version control functionality for workflows.

## Overview

The Git Version Control panel enables:

- Version history and tracking
- Branch management
- Commit and push workflows
- Rollback and restore

## Features

- **Version History** — view workflow version history
- **Branch Management** — create and manage branches
- **Commit Changes** — commit workflow changes
- **Rollback** — restore previous workflow versions
