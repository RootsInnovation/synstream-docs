# Sidebar - File Download

The File Download panel allows you to download files from workflows and the system.

## Overview

The File Download panel supports:

- Downloading workflow files
- Exporting workflow data
- Downloading execution results
- File export options

## Features

- **File Download** — download files from workflows
- **Data Export** — export workflow data
- **Result Download** — download execution results
- **Export Formats** — support multiple export formats
