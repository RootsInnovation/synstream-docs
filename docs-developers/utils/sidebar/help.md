# Sidebar - Help

The Help panel provides documentation, guides, and assistance for using Synstream.

- Tutorials and examples
- FAQ and troubleshooting
- Support resources

## Features

- **Documentation** — access comprehensive documentation
- **Tutorials** — step-by-step guides and tutorials
- **Examples** — real-world examples and use cases
