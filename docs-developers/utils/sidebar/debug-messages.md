# Sidebar - Debug Messages

The Debug Messages panel displays real-time debugging information and logs during workflow execution.

## Overview

The Debug Messages panel shows:

- Execution logs and messages
- Error messages and warnings
- Debug output from nodes
- System messages

## Features

- **Real-time Logging** — view logs as they are generated
- **Error Tracking** — monitor errors and warnings
- **Debug Output** — inspect node debug messages
- **Message Filtering** — filter messages by type and severity
