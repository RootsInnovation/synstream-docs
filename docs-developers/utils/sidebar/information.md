# Sidebar - Information

The Information panel in the sidebar provides essential details about the current workflow, nodes, and system status.

## Overview

The Information panel displays:

- Workflow metadata and properties
- Node information and configuration
- System status and health
- Execution statistics

## Features

- **Workflow Details** — view workflow name, description, and metadata
- **Node Information** — inspect node properties and settings
- **System Status** — monitor system health and performance
- **Statistics** — view execution statistics and metrics
