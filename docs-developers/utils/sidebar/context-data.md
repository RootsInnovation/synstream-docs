# Sidebar - Context Data

The Context Data panel displays workflow context and runtime data.

## Overview

The Context Data panel shows:

- Workflow context variables
- Runtime data and state
- Message payloads
- Environment variables

## Features

- **Context Variables** — view and edit context variables
- **Runtime Data** — inspect runtime data and state
- **Message Inspection** — examine message payloads
- **Environment Variables** — manage environment variables
