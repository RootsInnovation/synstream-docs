# Sidebar Features

Learn about sidebar features in the Synstream workflow designer. The sidebar provides access to nodes, search, and workflow management tools.

## Sidebar Components

### Node Library

The node library provides access to all available nodes:

- **Common Nodes** — basic workflow nodes
- **Extract and Load Nodes** — data extraction and loading nodes
- **Transform Nodes** — data transformation nodes
- **Advanced Nodes** — advanced workflow nodes

### Node Search

Search for nodes by:

- Node name
- Node category
- Node description
- Node functionality

### Node Categories

Nodes are organized into categories:

- **Common** — basic workflow nodes
- **Extract and Load** — data extraction and loading
- **Transform** — data transformation
- **Advanced** — advanced features

## Using the Sidebar

### Browsing Nodes

1. Open the sidebar
2. Browse node categories
3. Expand categories to see nodes
4. View node descriptions

### Searching Nodes

1. Use the search bar
2. Type node name or description
3. Filter results by category
4. Select node to view details

### Adding Nodes

1. Find node in sidebar
2. Drag node to canvas
3. Drop node on canvas
4. Configure node settings

## Sidebar Features

### Node Filtering

Filter nodes by:

- Category
- Functionality
- Usage
- Complexity

### Node Details

View node details:

- Node description
- Configuration options
- Input/output requirements
- Usage examples

### Node Favorites

Mark favorite nodes:

- Quick access to frequently used nodes
- Custom node collections
- Personal node library
