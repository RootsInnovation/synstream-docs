# System - Import

The Import functionality allows you to import workflows, configurations, and data into Synstream.

## Overview

Import supports:

- Importing workflows from files
- Importing configurations
- Importing data and resources
- Batch import operations

## Features

- **Workflow Import** — import workflows from files
- **Configuration Import** — import system configurations
- **Data Import** — import data and resources
- **Batch Import** — import multiple items at once
