# System - Export

The Export functionality allows you to export workflows, configurations, and data from Synstream.

## Overview

Export supports:

- Exporting workflows to files
- Exporting configurations
- Exporting data and resources
- Batch export operations

## Features

- **Workflow Export** — export workflows to files
- **Configuration Export** — export system configurations
- **Data Export** — export data and resources
- **Batch Export** — export multiple items at once
