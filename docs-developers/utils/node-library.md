# Node Library

Learn about the node library in the Synstream workflow designer sidebar.

## Overview

The node library provides access to all available nodes for building workflows. Nodes are organized into categories for easy navigation.

## Node Categories

### Common Nodes

Basic nodes for workflow control:

- Start
- Inject
- Debug
- Complete
- Catch
- Status
- Link In/Out
- Comment

### Extract and Load Nodes

Nodes for data extraction and loading:

- XLSX
- JSON
- CSV
- Database
- API
- File System

### Transform Nodes

Nodes for data transformation:

- Function
- MultiTask
- Hash
- Filter
- Sort
- Aggregate
- Map

### Advanced Nodes

Advanced nodes for complex operations:

- TimeTrace
- Validation
- Encryption
- Compression
- Scheduling

## Browsing Nodes

### Category Navigation

1. Open the sidebar
2. Expand node categories
3. Browse nodes in each category
4. View node descriptions

### Node Details

View node details by:

- Clicking on node name
- Viewing node description
- Checking configuration options
- Reading usage examples

## Adding Nodes

### Drag and Drop

1. Find node in library
2. Drag node to canvas
3. Drop node on canvas
4. Configure node settings

### Node Configuration

After adding a node:

1. Click on node to select
2. Configure node settings
3. Set input parameters
4. Configure output settings

## Node Information

### Node Description

Each node includes:

- Node name
- Node description
- Configuration options
- Input/output requirements
- Usage examples

### Node Status

Node status indicators:

- Available — node is available
- Recommended — node is recommended
- Deprecated — node is deprecated
- New — node is new
