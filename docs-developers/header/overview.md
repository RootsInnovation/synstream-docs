# Header Features

Learn about header features in the Synstream workflow designer. The header provides access to workflow management, deployment, and additional tools.

## Header Components

### Workflow Name

Display and edit workflow name:

- Current workflow name
- Edit workflow name
- Save workflow name
- Workflow status

### Use Case Tabs

Switch between different use cases:

- Multiple use cases per workflow
- Switch between use cases
- Create new use cases
- Delete use cases

### Deploy Button

Deploy workflow to production:

- Deploy workflow
- Select deployment environment
- Review deployment configuration
- Monitor deployment status

### Menu Button

Access additional features:

- Workflow settings
- Export workflow
- Import workflow
- Workflow history

### User Profile

Access user profile and settings:

- User profile
- User settings
- Logout
- Account management

## Using Header Features

### Workflow Management

1. Click on workflow name to edit
2. Change workflow name
3. Save changes
4. View workflow status

### Use Case Management

1. Click on use case tab to switch
2. Create new use case
3. Delete use case
4. Rename use case

### Deployment

1. Click Deploy button
2. Select deployment environment
3. Review deployment configuration
4. Deploy workflow

### Menu Access

1. Click Menu button
2. Access workflow settings
3. Export/import workflow
4. View workflow history

## Header Features

### Workflow Status

View workflow status:

- Ready — workflow is ready
- Running — workflow is running
- Error — workflow has errors
- Deployed — workflow is deployed

### Deployment Status

View deployment status:

- Pending — deployment pending
- In Progress — deployment in progress
- Success — deployment successful
- Failed — deployment failed

### Notification Center

Access notifications:

- Workflow notifications
- Deployment notifications
- Error notifications
- System notifications
