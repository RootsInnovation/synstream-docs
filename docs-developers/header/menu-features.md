# Menu Features

Learn about menu features in the Synstream workflow designer header.

## Overview

Menu features provide access to:

- Workflow settings
- Export/import workflows
- Workflow history
- Additional tools

## Menu Options

### Workflow Settings

Access workflow settings:

- Workflow name and description
- Execution schedule
- Error handling
- Notification settings
- Resource limits

### Export Workflow

Export workflow:

- Export workflow definition
- Export workflow configuration
- Export workflow data
- Share workflow with team

### Import Workflow

Import workflow:

- Import workflow definition
- Import workflow configuration
- Import workflow data
- Restore workflow from backup

### Workflow History

View workflow history:

- Previous workflow versions
- Change history
- Rollback options
- Version comparison

## Using Menu Features

### Accessing Menu

1. Click Menu button in header
2. Select menu option
3. Configure settings
4. Save changes

### Menu Navigation

Navigate menu options:

- Workflow Settings
- Export/Import
- Workflow History
- Additional Tools
