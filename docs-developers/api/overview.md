# API Reference Overview

The Synstream API provides programmatic access to all platform features. Use the API to manage products, pipelines, executions, and more.

## Authentication

All API requests require authentication using API keys or OAuth tokens.

## Base URL

```
https://api.synstream.com/v1
```

## Rate Limits

API requests are rate-limited to ensure fair usage. Check response headers for current rate limit status.

## API Endpoints

- **Products** — manage products and configurations
- **Pipelines** — create and manage pipeline flows
- **Executions** — monitor and manage pipeline executions
- **Users** — manage users and permissions
- **Webhooks** — configure webhook endpoints
