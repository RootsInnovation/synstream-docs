# Integration Overview

Learn how to integrate Synstream with your existing systems and applications.

## Integration Patterns

- **REST API Integration** — use the REST API to interact with Synstream
- **Webhook Integration** — receive events from Synstream pipelines
- **SDK Integration** — use official SDKs for popular languages
- **Custom Connectors** — build custom connectors for your systems

## Common Use Cases

- **CI/CD Integration** — integrate Synstream into your deployment pipeline
- **Monitoring Integration** — connect with monitoring and alerting systems
- **Data Pipeline Integration** — integrate with data processing systems
- **Authentication Integration** — integrate with identity providers
