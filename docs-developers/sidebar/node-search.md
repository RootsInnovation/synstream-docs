# Node Search

Learn how to search for nodes in the Synstream workflow designer sidebar.

## Overview

Node search helps you quickly find nodes by name, category, description, or functionality.

## Using Node Search

### Basic Search

1. Click on the search bar in the sidebar
2. Type node name or description
3. View search results
4. Select node to view details

### Advanced Search

Search by:

- **Node Name** — search by node name
- **Category** — filter by node category
- **Description** — search by node description
- **Functionality** — search by node functionality

## Search Features

### Search Filters

Filter search results by:

- Category
- Functionality
- Usage
- Complexity

### Search Suggestions

Get search suggestions:

- Autocomplete suggestions
- Related nodes
- Popular nodes
- Recently used nodes

### Search Results

Search results display:

- Node name
- Node category
- Node description
- Node status

## Search Examples

### Search by Name

**Input:**
```
xlsx
```

**Results:**
- XLSX Extract
- XLSX Load

### Search by Category

**Input:**
```
extract
```

**Results:**
- XLSX Extract
- JSON Extract
- CSV Extract
- Database Extract

### Search by Functionality

**Input:**
```
transform
```

**Results:**
- Function
- MultiTask
- Hash
- Filter
- Sort
- Aggregate
- Map

## Best Practices

- **Use Specific Terms** — use specific search terms for better results
- **Try Different Terms** — try different search terms if results are not helpful
- **Use Filters** — use filters to narrow down results
- **Check Suggestions** — check search suggestions for related nodes
