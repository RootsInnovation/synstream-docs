# Guides Overview

The guides collection documents practical workflows for teams building on Synstream. Each tutorial focuses on a real-world scenario so you can adapt it to your stack.

## How to Use These Guides

- **Choose a persona** — developer, operator, or analyst. Each guide highlights who benefits most.
- **Follow the walkthrough** — step-by-step instructions with CLI and UI actions.
- **Review the takeaways** — validation steps, metrics to monitor, and follow-up tasks.

Ready to go deeper? Jump straight to a guide that matches your use case or explore the Concepts section to understand the architecture behind each workflow.
