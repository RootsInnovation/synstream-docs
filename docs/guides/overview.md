# Overview

This guide collection walks you through the Synstream Console workflow. Follow these steps to set up your account, manage users and products, and start building automation flows.

## Console Workflow

The Synstream Console is your central management interface. Follow these steps in order:

1. **[Register and Login](./register-login)** — create your account and log into the Console
2. **[Users and Products Management](./users-products)** — manage users, licenses, and products in the Console
3. **[Create Product](./create-product)** — create a new product in the Console
4. **[Enter Product](./enter-product)** — access your product's Pipeline Studio from the Console
